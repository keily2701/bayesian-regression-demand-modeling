# Academic Version

This folder contains the academic version of the work that motivated this project.

The material corresponds to an undergraduate graduation thesis developed as part of a Bachelor's degree in Mathematics, with a specialization in Statistics and Applied Mathematics.

The academic version includes formal theoretical background, methodological justification, and statistical foundations required for graduation purposes.

## Contents

- `tesis.pdf`: Original undergraduate thesis submitted for the completion of the Bachelor's degree in Mathematics.

## Note

The main repository presents an applied and simplified adaptation of this work, focusing on data analysis, statistical modeling, and practical insights oriented toward real-world decision-making.

Readers interested in business or applied perspectives are encouraged to consult the main README and analysis notebooks.
